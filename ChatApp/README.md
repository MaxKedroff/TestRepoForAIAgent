# Multi Chat на ASP.NET

Веб-приложение для многопользовательского чата с:
- ручной учеткой администратора;
- выдачей пользовательских аккаунтов через никнейм и длинный токен;
- созданием чатов только админом;
- доступом пользователя только к своим чатам;
- обменом сообщениями в реальном времени (SignalR);
- дополнительной фичей: индикатор "печатает...".

## Запуск

1. Установите .NET 8 SDK.
2. Перейдите в папку приложения:
   ```bash
   cd ChatApp
   ```
3. Запустите приложение:
   ```bash
   dotnet run
   ```
4. Откройте URL из консоли.

## Вход

- Админ: `/account/admin-login`
  - Логин/пароль задаются в `appsettings.json` (`Admin` секция).
- Пользователь: `/account/login`
  - Использует токен, который выдал админ.
